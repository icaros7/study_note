 ## 파이프라인 속도 구하기
 ```
단일 연산 소요 시간 = 클록 주기 * 명령어 개수

걸리는 시간 T = 파이프라인 단계 K + (명령어 개수 N - 1)

속도 향상 Sp = (K * N) / {K + (N - 1)}

단계 별 속도: 단계 별 명령어 개수 나누고, 해당 파이프라인 단계에 따라 계산 후 모두 더하기
 ```

 ## 슈퍼스칼라 속도 계산식

```
전체 시간 T(m) = 파이프라인 단계 수 K + (명령어 수 N - 슈퍼스칼라way **m**) / m

Sp = T(1) / T(m) = (k + N - 1) / (K + (n - m) / m)
```
